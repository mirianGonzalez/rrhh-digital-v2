1. Introducción

Bienvenido al Sistema de Gestión de Legajos RRHH DIGITAL del Honorable Concejo Deliberante de Posadas.

Este sistema permite la gestión integral de los legajos del personal, incluyendo:

  • Alta y modificación de legajos
  • Gestión de documentos obligatorios
  • Generación de Declaraciones Juradas anuales
  • Auditoría completa de todas las acciones
  • Backup automático y seguridad
Versión: 2.0 Profesional - Marzo 2026

3. Gestión de Legajos

3.1 Crear Nuevo Legajo

  1. Desde el panel principal, haga clic en "Nuevo Legajo"
  2. Complete los datos personales del agente
  3. Complete los datos laborales
  4. Haga clic en "Guardar Legajo"

El sistema creará automáticamente la carpeta del agente y registrará la acción en auditoría.

3.2 Ver Detalle del Legajo

Al ingresar a un legajo, encontrará:

  • Información personal: Datos del agente
  • Progreso: Barra de avance de documentos
  • Documentos obligatorios: Organizados por categoría
  • DDJJ: Historial de declaraciones juradas

3.3 Documentos Obligatorios

El sistema gestiona automáticamente los siguientes documentos:

Identidad:

  • DNI Frente y Dorso
  • Partida de Nacimiento
  • CUIL
  • Foto 4x4

Familia:

  • Partida de Matrimonio
  • Partidas de Nacimiento de hijos

Educación:

  • Título Secundario
  • Título Terciario/Universitario
  • Certificado Analítico

Laboral:

  • Contrato/Decreto
  • Seguro de Vida
  • Certificado de Servicios
Los documentos marcados con * son obligatorios para generar la DDJJ.

4. Declaraciones Juradas

4.1 Nueva Declaración Jurada

Requisitos previos:

  • Todos los documentos obligatorios deben estar cargados
  • El legajo debe estar en estado "Activo"

Para generar una nueva DDJJ:

  1. Ir al detalle del legajo
  2. Hacer clic en "Nueva DDJJ"
  3. Completar el formulario con los datos actualizados
  4. El sistema generará automáticamente el PDF con código QR

4.2 Validación de DDJJ

Cada DDJJ generada incluye:

  • Código de validación único de 16 caracteres
  • Código QR para verificación rápida
  • Hash SHA256 para integridad documental
  • Sello digital del organismo

Para validar una DDJJ:

  1. Escanee el código QR con cualquier lector
  2. O ingrese a: http://sistema/validar/CODIGO
  3. El sistema mostrará los datos de la DDJJ y su estado

5. Auditoría

El sistema registra automáticamente todas las acciones:

  • Accesos al sistema (login/logout)
  • Creación/modificación de legajos
  • Subida de documentos
  • Generación de DDJJ
  • Intentos fallidos de acceso

Los registros incluyen:

  • Usuario y rol
  • Fecha y hora exacta
  • Dirección IP
  • Detalle de la acción
  • Hash de documentos (cuando corresponde)

6. Seguridad y Backup

Backup Automático

  • Backup diario a las 23:00 hs
  • Backup manual disponible para administradores
  • Retención de 30 días
  • Hash de verificación para integridad

Seguridad

  • Contraseñas hasheadas con bcrypt
  • Bloqueo por 5 intentos fallidos
  • Sesión segura con timeout de 8 horas
  • HTTPS requerido en producción
  • Auditoría de accesos

7. Preguntas Frecuentes

¿Qué hago si olvidé mi contraseña?

Contacte al administrador del sistema para resetear su contraseña.

¿Puedo subir varios documentos a la vez?

Actualmente se deben subir uno por uno para mantener el control de versiones.

¿Los documentos se convierten automáticamente a PDF?

Sí, el sistema convierte automáticamente imágenes y documentos de Word a PDF.

¿Cómo sé si un legajo está completo?

El sistema muestra una barra de progreso y el estado de cada documento obligatorio.

¿Puedo modificar una DDJJ ya finalizada?

No, las DDJJ finalizadas son inmodificables. Debe generar una nueva versión.
